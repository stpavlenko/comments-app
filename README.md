# Comments app

Live comments using Vue.js and SSE

http://vue-app-pavlenko.std-2121.ist.mospolytech.ru/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
